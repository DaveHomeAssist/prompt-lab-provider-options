// core loop

Built around the way serious prompt work actually happens.

Prompt Lab is organized around a simple cycle: engineer a strong draft, test the result across models or variants, then iterate with saved structure instead of rewriting from scratch every time.

Engineer

Build prompts with structure.

Draft in the editor, assemble reusable blocks in the composer, fill variables at run time, and keep your high-value prompts in a searchable library.

Test

Compare output instead of trusting vibes.

Run A/B experiments, inspect differences across providers, and use history to see what changed instead of relying on memory.

Iterate

Keep the wins and refine the rest.

Version saved prompts, restore old drafts, and keep a local-first workflow that lets you iterate quickly without turning the tool into overhead.

// capabilities

The working set for prompt teams and solo operators.

Prompt Lab is not trying to be a general AI dashboard. It focuses on the real workflow around prompts: writing, enhancing, comparing, saving, and reusing them across providers.

Enhance

Refine the draft fast.

Use provider-backed enhancement to tighten role clarity, constraints, format, and intent without rebuilding the prompt by hand.

Compare

A/B test prompt variants.

Run two approaches side by side and inspect where they diverge in tone, completeness, and usefulness.

Library

Keep reusable prompts organized.

Save prompts with metadata, restore older versions, and move proven blocks into future drafts instead of repeating work.

Compose

Assemble prompts from modules.

Chain instructions, personas, and constraints into structured outputs that are easier to reuse and easier to test.

Variables

Reuse one prompt across many contexts.

Fill values at run time with variable placeholders so the same template can adapt to different tasks, projects, or inputs.

Local-first

Keep settings in your control.

Prompt state lives locally, with direct provider calls in extension and desktop mode and a narrow proxy path only for the hosted web shell.
